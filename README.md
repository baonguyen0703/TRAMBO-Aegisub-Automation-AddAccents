#### Installation
  - Put "Trambo.AddAccents.lua" into autoload folder, the default path is:  
    - Aegisub 64-bit: **C:\Program Files\Aegisub\automation\autoload**  
    - Aegisub 32-bit: **C:\Program Files (x86)\Aegisub\automation\autoload**
